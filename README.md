# @xiaheng/img-cli

图片更换尺寸命令工具。

## 使用说明

### 安装

`npm i @xiaheng/img-cli -g`

### 使用

`img-cli`

## 开发指引

`git clone https://github.com/15380831711/img-cli.git`

`cd img-cli`

`npm i`

`node .\bin\xa-img.js`

`npm run test`
